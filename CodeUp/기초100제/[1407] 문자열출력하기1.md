# CodeUp 기초 100제

## [1407]: 문자열 출력하기 1

### 🌴 문제

길이(글자수)가 100이하인 문자열을 입력받아 공백을 제거하고 출력하시오.

#### ◻ 입력

문자열이 입력된다.(글자 수는 100글자 이하이고, 알파벳 대소문자와 공백 문자만 입력된다.)

#### ◻ 출력

공백을 제거한 후 출력한다.

---

### 🤠 MY code

```c++
#include <stdio.h>

int main() {
    int i;
    char a[101];
    gets(a);
    for(i = 0; a[i] != '\0'; i++) {
        if (a[i] != ' ') printf("%c", a[i]);
    }
    return 0;
}

```
