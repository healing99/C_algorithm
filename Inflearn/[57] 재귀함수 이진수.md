# 인프런 알고리즘 문제풀이(C/C++)

## [57]: 재귀함수 이진수 출력

### 🌴 문제

10진수 N이 입력되면 2진수로 변환하여 출력하는 프로그램을 작성하세요. 단 재귀함수를 이용해서 출력해야 합니다.

#### ◻ 입력

첫 번째 줄에 10진수 N(1<=N<=1,000)이 주어집니다.<br>
입력예시 : 11

#### ◻ 출력

첫 번째 줄에 이진수를 출력하세요.<br>
출력예시 : 1011

---

### 🤠 MY code

```c++
#include <stdio.h>

void func(int x) {
	if (x ==0) return;
	func(x / 2);
	printf("%d", x % 2);
}

int main() {
	int n;
	scanf("%d", &n);
	func(n);
}
```
